# A 股交易佣金成本详解

投资的核心原则之一是**降低交易成本**，成本的高低直接影响最终收益。A 股交易的显性成本均会在券商交割单中列明，具体分为三类：

## 一、印花税



* **收取方**：政府

* **费率**：卖出时按成交金额的**千分之 0.5**收取（2023 年 8 月 28 日从千分之 1 下调）

* **特点**：单边收取（仅卖出）、无起步价，按实际成交金额计算，哪怕 1 分钱也按比例收取。

## 二、过户费



* **收取方**：中国证券登记结算公司

* **费率**：沪深两市买卖双向收取，按成交金额的**万分之 0.1**（2022 年 4 月 29 日从万分之 0.2 下调）

* **特点**：无起步价，统一费率，所有券商和投资者标准一致。

## 三、券商佣金

上述的印花税和过户费是无法避免的，每位投资者都需要缴纳，而佣金则是唯一可与券商协商的费用，分为**全佣**和**净佣金**两类，核心构成及费率规则如下：

### 1. 佣金核心构成（买卖双边收取）



| 费用类型            | 具体项目       | 费率                | 归属方  |
| --------------- | ---------- | ----------------- | ---- |
| **规费（固定）**      | 证管费        | 万分之 0.2           | 证监会  |
|                 | 证券交易经手费    | 万分之 0.341         | 交易所  |
|                 | **规费合计**   | **万分之 0.541**     | -    |
| **券商净佣金**       | 券商服务费      | 万分之 0.313（行业常见标准） | 证券公司 |
| **全佣（投资者最终支付）** | 规费 + 券商净佣金 | **万分之 0.854**     | -    |

### 2. 全佣与净佣金的区别



| 佣金类型 | 包含内容       | 交割单显示形式           | 投资者注意事项                |
| ---- | ---------- | ----------------- | ---------------------- |
| 全佣   | 规费 + 券商净佣金 | 合并为 “佣金” 一项       | 市场主流报价口径，直接对应投资者总成本    |
| 净佣金  | 仅券商服务费     | 分 “净佣金” 和 “规费” 两项 | 需叠加规费才是实际交易成本，对比费率时要换算 |

### 3. “免 5” 规则与实际费用计算



* **“免 5” 含义**：佣金按实际金额收取，无最低 5 元门槛；非免 5 则不足 5 元按 5 元收取。

* **举例计算**：成交金额 1 万元，佣金率为万 0.854



| 规则   | 理论佣金（1 万 × 万 0.854） | 实收佣金    | 差额         |
| ---- | ------------------- | ------- | ---------- |
| 免 5  | 0.854 元             | 0.854 元 | 节省 4.146 元 |
| 非免 5 | 0.854 元             | 5 元     | 多付 4.146 元 |

### 4. 非整数费率 “万 0.854” 的由来

2023 年 8 月 28 日交易所下调**证券交易经手费**（从万 0.487 降至万 0.341），且要求券商不得截留降费红利。

调整前后的费率逻辑对比：



| 时间节点        | 规费合计      | 券商净佣金（不变） | 投资者全佣费率         |
| ----------- | --------- | --------- | --------------- |
| 2023.8.28 前 | 万分之 0.687 | 万分之 0.313 | 万分之 1（即 “万一免五”） |
| 2023.8.28 后 | 万分之 0.541 | 万分之 0.313 | 万分之 0.854       |

## 四、佣金能降到 0 吗？

答案是：**不现实**。

券商收取的万 0.313 净佣金，需覆盖多项刚性成本：



1. 交易所席位费：每个席位年费 60 万，券商需多个席位保障交易流畅；

2. 投资者保护基金：按券商营业收入的 0.5%-5% 缴纳；

3. 委托流量费：每笔委托（无论成交、撤单）收取 0.1 元；

4. 交易系统运维费：包括 APP、服务器、通信网络等技术投入。

目前 “万 0.854 免 5” 已是行业低佣底线，券商利润微薄。若佣金过低，可能导致服务缩水（如交易卡顿、客服响应慢）。

## 总结

一味追求极致低佣不可取，合理的佣金能保障券商提供稳定服务。聪明的投资者应平衡**成本**与**服务质量**，选择长期可靠的合作券商。
